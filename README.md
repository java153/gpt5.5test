# Neon Key Playground

A lightweight browser rhythm playground designed to run smoothly on older laptops (like a 2017 MacBook).

## Run

Just open `index.html` in a browser (Chrome, Edge, Safari, or Firefox).

## Controls

- Play keys: `W E D J I O`
- Click **Start / Unmute** once to enable audio.
- Change BPM and Glow sliders.

## GPT Image 2 Art Integration

The game will auto-load PNG icons from:

- `assets/gpt/w.png`
- `assets/gpt/e.png`
- `assets/gpt/d.png`
- `assets/gpt/j.png`
- `assets/gpt/i.png`
- `assets/gpt/o.png`

If images are missing, the game uses simple neon placeholders.

### Suggested GPT Image 2 prompt

"Tiny 2010 indie game icon, flat colors, pixel-ish edges, transparent background, neon palette, symbol for key W"

(Repeat for E, D, J, I, O.)
